Useful Shortcuts:
=================
- Right Tab - ctrl + R
- Left Tab - ctrl + E
- New Tab - shift + T / ctrl + T
- Left Tab (Tree) - ctrl + A
- Right Tab (Code) - ctrl + D
- Show Hidden Files: shift + I
- Undo - ctrl + Z
- Redo - ctrl + Y
- Goto line: :<line_number> (ex: :12)
- When having several terminals open, toggle: alt + 1/2/3/...
- F2: show current file in node tree
